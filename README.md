# GeoSpatial Site Readiness Analyzer (GSRA)

AI-powered system to evaluate site suitability across Gujarat using geospatial data, ML, and spatial analysis.

## Tech Stack
- FastAPI (Backend)
- PostgreSQL + PostGIS (Database)
- React + MapLibre (Frontend)
- OSRM (Routing Engine)
- PyTorch (ML)

## Project Structure
- backend/ → API + scoring engine
- frontend/ → map UI
- data/ → raw + processed geospatial data
